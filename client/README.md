# Chat Client (Vite + React + Tailwind)

## Setup

1. Edit `client/.env` (optional) and set `VITE_BACKEND_URL` if your backend is not at http://localhost:4000

2. Install dependencies:
```bash
cd client
npm install
```

3. Run dev:
```bash
npm run dev
```

Open the Vite dev server (usually http://localhost:5173).
